# Artifact Inventory

Large data files, generated figures, local Hugging Face dataset snapshots, and embedding arrays are excluded from Git. This file records the expected project-relative artifacts used by the BioNLP 2025 analyses.

## Embeddings

`embeddings/` should contain one `.npz` file per clinical category. Each file contains:

```text
embeddings: float32 array with shape (n_cases, 384)
topic: object array
age: object array
sex: object array
```

Expected files:

```text
CVS_embeddings.npz
DERM_embeddings.npz
EENT_embeddings.npz
ENDO_embeddings.npz
GI_embeddings.npz
GU_embeddings.npz
History_embeddings.npz
Lab_Image_embeddings.npz
LYMPH_embeddings.npz
MSK_embeddings.npz
Neuro_embeddings.npz
Pregnancy_embeddings.npz
RESP_embeddings.npz
Vitals_Hema_embeddings.npz
```

These files are generated by `notebooks/pool_mean_embeddings.ipynb` and consumed by `notebooks/faiss_retrieval.ipynb`.
